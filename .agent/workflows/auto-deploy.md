# Git Commit & Deploy Workflow

Usa este workflow inmediatamente después de terminar cambios efectivos para guardarlos y redesplegar a producción.

// turbo-all
```bash
git add .
git commit -a -m "chore(auto): apply task changes, push to main and redeploy origin"
git push origin main
```
